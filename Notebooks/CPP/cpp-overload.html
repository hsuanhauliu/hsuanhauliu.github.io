Overloading in C++

Date: 10/11/2017

Overloading is a feature that allows you to have multiple functions that share the same name. This means that you can use the same function to deal with inputs with different number of parameters, or inputs with different data types. Operators such as <<, >>, = can also be overloaded.


Function overloading example:

// Have a function called timesTwo to deal with float and int type input.

float timesTwo(float num) {
   	return num * 2;
}

int timesTwo(int num) {
   	return num * 2;
}

Operator overloading example:

#include <iostream>
#define tab '\t'

using namespace std;

struct data {
	string name;
	int age;
	float height;
};

// Overloaded insertion operator for data struct
ostream& operator << (ostream& str_out, struct data& d) {
	str_out << d.name << tab;
	str_out << d.age << tab;
	str_out << d.height << endl;
	return str_out;
}
