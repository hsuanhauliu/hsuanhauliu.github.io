Recursion

Date: 10/11/2017

Recursion is a useful programming technique, where a function calls itself repeatedly until it reaches the base case. It is usually used to solve problems that can be broken down into smaller pieces. This clever idea allows us to write elegant codes to solve problems. The idea is simple, but the implementation can be a little difficult at first. Don't panic. You'll get the hang of it after practicing it a few times!


Here is a simple example of a recursive algorithm.

// Use recursion to calculate factorial
long long int factorial(int num) {
	long long int sum = num;

	// base case
	if (num == 0)
		return 0; // factorial of 0 is 1
	else if (num == 1)
		return 1; // return 1 when it reaches 1
	else
		sum *= factorial(--num); // decrement by one then pass it into the function

	return sum;
}
