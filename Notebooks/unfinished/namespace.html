Recurrsion

Date: 10/8/2017

Recurrence is a useful programming technique that allows a function to call itself. This technique can be very powerful if used correctly. This clever idea helps the programmer to save time and memory allocation. The idea is simple, but the implementation can be difficult to grasp at fist. This technique is heavily used in functional language, though good programmers do need to learn how to utilize this technique in OOP as well 


// Use recurrence to calculate the fibbinaci number.
int Fibbinaci(int num) {
	int sum = num;

	// base case
	if (num == 1)
		return 1; // return 1 when it reaches 1
	else
		sum *= Fibbinaci(--num); // decrement by one then pass it into the function

	return sum;
}
